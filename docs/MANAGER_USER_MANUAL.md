## Introduction

RentFlow Manager is a comprehensive property management platform designed to help you efficiently manage your properties, tenants, leases, and financial operations. This manual will guide you through all the features and functionalities available to managers.

### Key Features
- Property and unit management
- Tenant onboarding and management
- Lease creation and management
- Automated rent collection and payment processing
- Maintenance request tracking
- Financial reporting and analytics
- Team and submanager management
- Communication tools

## Getting Started

### Account Setup

1. **Login to Manager Dashboard**
   - Navigate to https://app.rentflow.ink
   - Enter your email and password
   - Click "Sign In"

2. **Complete Onboarding**
   - Follow the onboarding wizard to set up your profile
   - Configure your business information
   - Set up payment methods
   - Add your first property

### Navigation

The Manager Dashboard is organized into the following sections:

- **Overview** - Dashboard with key metrics and quick actions
- **Properties** - Manage your properties and units
- **Tenants** - Manage tenant information and activities
- **Leases** - Create and manage lease agreements

- **Reports** - Access financial and operational reports
- **Settings** - Configure your account and preferences

## Property Management

### Adding a Property

1. Navigate to **Properties** → **Add Property**
2. Fill in property details:
   - Property name
   - Address
   - Property type (apartment, house, commercial, etc.)
   - Number of units
   - Property description
3. Upload property photos
4. Set property amenities
5. Click **Save Property**

### Managing Units

1. Navigate to **Properties** → Select a property
2. Click **Manage Units**
3. Add units with details:
   - Unit number/name
   - Unit type (studio, 1-bedroom, 2-bedroom, etc.)
   - Monthly rent amount
   - Deposit amount
   - Square footage
   - Amenities
4. Set unit status (available, occupied, maintenance)

### Property Analytics

View property-specific analytics:
- Occupancy rates
- Revenue trends
- Maintenance costs
- Tenant turnover

## Tenant Management

## Lease Management

### Creating a Lease

1. Navigate to **Leases** → **Create Lease**
2. Select property and unit
3. Select tenant
4. Set lease terms:
   - Start date
   - End date
   - Monthly rent amount
   - Deposit amount
   - Payment due date
   - Late fee policy
5. Add lease terms and conditions
6. Upload signed agreement
7. Click **Create Lease**

### Lease Renewals

1. Navigate to **Leases** → Select expiring lease
2. Click **Renew Lease**
3. Update lease terms
4. Send renewal notice to tenant
5. Get tenant acknowledgment
6. Save renewed lease

### Lease Termination

1. Navigate to **Leases** → Select lease to terminate
2. Click **Terminate Lease**
3. Select termination reason
4. Set termination date
5. Calculate final settlement
6. Send termination notice
7. Process deposit refund

## Rent Collection

### Payment Methods

RentFlow supports multiple payment methods:
- **M-Pesa** - Mobile money payments
- **Bank Transfer** - Direct bank transfers
- **Credit/Debit Card** - Card payments
- **Cash** - Manual cash payments

### Setting Up M-Pesa

1. Navigate to **Settings** → **Payment Methods**
2. Select **M-Pesa**
3. Enter your M-Pesa business shortcode
4. Configure M-Pesa API credentials
5. Set up payment notifications
6. Click **Save**

### Sending Rent Reminders

1. Navigate to **Billing** → **Rent Reminders**
2. Select tenants to remind
3. Choose reminder method (SMS, email, or both)
4. Customize reminder message
5. Click **Send Reminders**

### Viewing Payment History

1. Navigate to **Billing** → **Payment History**
2. Filter by date range, property, or tenant
3. View payment details
4. Download payment reports

### Handling Late Payments

1. Navigate to **Billing** → **Late Payments**
2. View overdue payments
3. Send late payment notices
4. Apply late fees if applicable
5. Track payment follow-ups

## Troubleshooting

### Common Issues

#### Login Issues
- **Problem**: Cannot login to dashboard
- **Solution**: 
  - Verify email and password are correct
  - Reset password if needed
  - Clear browser cache
  - Try different browser

#### Payment Processing Issues
- **Problem**: M-Pesa payments not processing
- **Solution**:
  - Verify M-Pesa credentials
  - Check API status
  - Verify tenant phone number
  - Contact support if issue persists

#### Maintenance Request Issues
- **Problem**: Cannot submit maintenance request
- **Solution**:
  - Verify property and unit are selected
  - Check internet connection
  - Try refreshing the page
  - Contact support if issue persists

#### Report Generation Issues
- **Problem**: Reports not generating
- **Solution**:
  - Verify date range is valid
  - Check data availability
  - Try smaller date range
  - Contact support if issue persists

### Glossary

- **Unit**: Individual rental space within a property
- **Lease**: Legal agreement between landlord and tenant
- **Occupancy Rate**: Percentage of units currently occupied
- **Late Fee**: Additional charge for late rent payments
- **Deposit**: Security deposit held for potential damages
