## Getting Started

### Account Setup

1. **Login to Tenant Portal**
   - Navigate to https://app.rentflow.ink/tenant/login
   - Enter your email and password
   - Click "Sign In"

2. **First-Time Setup**
   - Verify your email address
   - Set up your password
   - Complete your profile
   - Set up payment methods

### Setting Up Payment Methods

RentFlow supports multiple payment methods:

#### M-Pesa Setup
1. Navigate to **Profile** → **Payment Methods**
2. Click **Add M-Pesa**
3. Enter your M-Pesa phone number
4. Verify your number via SMS
5. Click **Save**

#### Bank Transfer Setup
1. Navigate to **Profile** → **Payment Methods**
2. Click **Add Bank Account**
3. Enter bank details:
   - Bank name
   - Account number
   - Account holder name
4. Click **Save**

#### Card Payment Setup
1. Navigate to **Profile** → **Payment Methods**
2. Click **Add Card**
3. Enter card details:
   - Card number
   - Expiry date
   - CVV
   - Cardholder name
4. Click **Save**

## Rent Payments

### Making a Payment

1. Navigate to **Payments** → **Pay Rent**
2. Select payment method:
   - M-Pesa
   - Bank Transfer
   - Credit/Debit Card
3. Enter payment amount
4. Review payment details
5. Click **Pay Now**
6. Complete payment verification
7. Receive payment confirmation

### M-Pesa Payment

1. Select **M-Pesa** as payment method
2. Enter amount to pay
3. Click **Pay with M-Pesa**
4. You will receive an M-Pesa STK push on your phone
5. Enter your M-Pesa PIN to authorize payment
6. Receive payment confirmation

### Bank Transfer Payment

1. Select **Bank Transfer** as payment method
2. Enter amount to pay
3. Click **Generate Payment Reference**
4. Copy the payment reference number
5. Transfer funds to the provided bank account
6. Include the reference number in the transfer
7. Payment will be automatically matched and confirmed

### Card Payment

1. Select **Card** as payment method
2. Enter amount to pay
3. Enter card details or select saved card
4. Click **Pay Now**
5. Complete 3D Secure verification if required
6. Receive payment confirmation

### Payment History

View your payment history:
1. Navigate to **Payments** → **Payment History**
2. View all payments:
   - Payment date
   - Payment amount
   - Payment method
   - Payment status
3. Download payment receipts
4. Filter by date range

### Payment Reminders

- Automatic email reminders before due date
- SMS reminders (if enabled)
- Dashboard notifications
- Mobile app push notifications

### Late Payments

If payment is late:
- Late fee may be applied (as per lease terms)
- You will receive late payment notice
- Contact property management if you need assistance

### Payment Issues

If you encounter payment issues:
- Check your payment method is valid
- Verify sufficient funds
- Try alternative payment method
- Contact property management for assistance

## Troubleshooting

### Common Issues

#### Login Issues
- **Problem**: Cannot login to tenant portal
- **Solution**:
  - Verify email and password are correct
  - Reset password if needed
  - Clear browser cache
  - Try different browser
  - Contact property manager if account is locked

#### Payment Issues
- **Problem**: M-Pesa payment not processing
- **Solution**:
  - Verify M-Pesa number is correct
  - Check M-Pesa account has sufficient funds
  - Ensure phone has network coverage
  - Try again after a few minutes
  - Contact property manager if issue persists

- **Problem**: Bank transfer not recognized
- **Solution**:
  - Verify payment reference number is correct
  - Ensure transfer was completed
  - Wait 1-2 business days for processing
  - Contact property manager with transfer confirmation

#### Maintenance Request Issues
- **Problem**: Cannot submit maintenance request
- **Solution**:
  - Verify all required fields are filled
  - Check internet connection
  - Try refreshing the page
  - Contact property manager if issue persists

#### Notification Issues
- **Problem**: Not receiving notifications
- **Solution**:
  - Check notification preferences
  - Verify email address is correct
  - Check spam/junk folder
  - Ensure phone number is correct for SMS
  - Contact property manager if issue persists
