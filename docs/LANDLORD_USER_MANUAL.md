## Introduction

RentFlow Landlord Portal is a dedicated platform for property owners to monitor their properties, track revenue, and view financial performance. This manual will guide you through all the features available to landlords.

### Key Features
- Property performance monitoring
- Revenue tracking and analytics
- Lease overview and status
- Financial reporting
- Payout management
- Property insights

### Important Note
As a landlord, you have a revenue-only view of your properties. You will not see individual tenant names, contact information, or payment details to protect tenant privacy. All tenant data is managed by your property manager.

## Getting Started

### Account Setup

1. **Login to Landlord Dashboard**
   - Navigate to https://app.rentflow.ink/landlord/login
   - Enter your email and password
   - Click "Sign In"

2. **Complete Profile Setup**
   - Verify your contact information
   - Set up payout preferences
   - Configure notification settings

### Navigation

The Landlord Dashboard is organized into the following sections:

- **Overview** - Dashboard with key metrics and property performance
- **Properties** - View your properties and their performance
- **Revenue** - Track revenue and payouts
- **Leases** - Overview of lease status and expirations
- **Reports** - Access financial reports
- **Settings** - Configure your account and preferences

## Revenue Management

### Viewing Revenue

1. Navigate to **Revenue**
2. View revenue by:
   - Time period (daily, weekly, monthly, yearly)
   - Property
   - Revenue type (rent, fees, other)

### Revenue Breakdown

- **Rent Revenue** - Income from rent payments
- **Late Fees** - Income from late payment fees
- **Other Fees** - Other miscellaneous fees
- **Total Revenue** - Sum of all revenue sources

### Payout Management

1. Navigate to **Revenue** → **Payouts**
2. View payout history
3. View pending payouts
4. Configure payout preferences:
   - Bank account details
   - Payout frequency (weekly, monthly)
   - Minimum payout amount
5. Request payout

### Payout Schedule

- **Weekly Payouts** - Processed every Friday
- **Monthly Payouts** - Processed on the last business day of the month
- **Minimum Payout** - Minimum amount required for automatic payout

### Revenue Analytics

View revenue analytics:
- Revenue trends over time
- Revenue by property
- Revenue growth rate
- Revenue forecasting
- Comparison with previous periods

## Financial Reports

### Available Reports

- **Revenue Report** - Detailed revenue breakdown
- **Profit & Loss Statement** - Income and expenses
- **Cash Flow Statement** - Cash flow analysis
- **Property Performance Report** - Individual property performance
- **Payout Report** - Payout history and status

### Generating Reports

1. Navigate to **Reports**
2. Select report type
3. Set filters:
   - Date range
   - Properties to include
   - Report format (PDF, Excel, CSV)
4. Click **Generate Report**
5. View or download report

### Scheduled Reports

1. Navigate to **Reports** → **Scheduled Reports**
2. Click **Schedule Report**
3. Select report type
4. Set frequency (weekly, monthly, quarterly)
5. Set delivery method (email, dashboard)
6. Click **Schedule**

### Report Analytics

View report analytics:
- Revenue trends
- Expense trends
- Profit margins
- Cash flow analysis
- Property performance comparison

## Troubleshooting

### Common Issues

#### Login Issues
- **Problem**: Cannot login to landlord dashboard
- **Solution**:
  - Verify email and password are correct
  - Reset password if needed
  - Clear browser cache
  - Try different browser
  - Contact property manager if account is locked

#### Payout Issues
- **Problem**: Not receiving payouts
- **Solution**:
  - Verify bank account details are correct
  - Check minimum payout amount
  - Verify payout frequency settings
  - Contact property manager if issue persists

#### Revenue Display Issues
- **Problem**: Revenue not displaying correctly
- **Solution**:
  - Refresh the page
  - Check date range filters
  - Verify property selection
  - Contact property manager if issue persists

#### Report Generation Issues
- **Problem**: Reports not generating
- **Solution**:
  - Verify date range is valid
  - Check data availability
  - Try smaller date range
  - Contact property manager if issue persists

## Privacy Notice

As a landlord, you have access to aggregate financial and operational data about your properties. Individual tenant information (names, contact details, payment history) is protected and only accessible to your property manager. This ensures tenant privacy while providing you with the insights needed to manage your properties effectively.

- **Net Income**: Revenue minus expenses
- **Property Performance**: Overall financial and operational performance of a property
