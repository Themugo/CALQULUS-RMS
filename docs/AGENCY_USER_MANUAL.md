## Introduction

RentFlow Agency Dashboard is a dedicated platform for property management agencies that operate on behalf of landlords. As an agency, you manage properties for multiple landlords, collect rent, handle maintenance, and earn commission on your services.

### Key Features
- Multi-landlord property management
- Commission-based revenue model
- Landlord relationship management
- Full tenant management capabilities
- Revenue sharing with landlords
- Agency-specific reporting
- Team management for agency staff

### Agency vs Manager

While agencies and managers both manage properties, key differences include:
- **Agencies** manage properties ON BEHALF OF landlords with commission-based revenue
- **Managers** manage properties directly with subscription-based revenue
- **Agencies** have revenue sharing configurations with landlords
- **Agencies** can collect rent to agency accounts or pass through to landlords

### Access Your Dashboard
- **URL**: https://app.rentflow.ink/agency/login
- **Role**: Agency Team Manager

## Property Management

### Adding a Property

1. Navigate to **Properties** → **Add Property**
2. Fill in property details:
   - Property name
   - Address
   - Property type (apartment, house, commercial, etc.)
   - Number of units
   - Property description
3. **Link to Landlord**:
   - Select existing landlord
   - Or add new landlord
   - Set revenue sharing percentage
   - Set commission percentage
   - Set payment destination (agency or landlord)
4. Upload property photos
5. Set property amenities
6. Click **Save Property**

1. Navigate to **Properties** → Select a property
2. Click **Manage Units**
3. Add units with details:
   - Unit number/name
   - Unit type (studio, 1-bedroom, 2-bedroom, etc.)

### Property Analytics

View property-specific analytics:
- Occupancy rates
- Revenue trends
- Maintenance costs
- Tenant turnover
- Commission earned
- Landlord payout

### Revenue Sharing Configuration

For each property, configure:
- **Revenue Share %**: Percentage of rent that goes to landlord
- **Commission %**: Percentage that agency keeps as commission
- **Payment Destination**: Where rent is collected (agency account or landlord account)
- **Operating Model**: Full management or collection-only

## Commission Management

### Commission Structure

Configure commission structure:
- **Default Commission %**: Standard commission rate
- **Property-Specific %**: Custom commission per property
- **Landlord-Specific %**: Custom commission per landlord
- **Volume Discounts**: Reduced commission for high-volume landlords

### Revenue Sharing Models

1. **Full Management Model**
   - Agency collects all rent
   - Agency pays landlord their share
   - Agency keeps commission
   - Agency handles all operations

2. **Collection-Only Model**
   - Agency collects rent
   - Rent goes directly to landlord
   - Agency invoices landlord for commission
   - Landlord handles operations

3. **Hybrid Model**
   - Some rent to agency, some to landlord
   - Commission calculated on agency portion
   - Flexible arrangement per landlord

### Commission Calculation

Commission is calculated as:
```
Commission = Rent Amount × Commission Percentage
Landlord Share = Rent Amount - Commission
```

View commission analytics:
- Commission by property

- Commission trends

## Troubleshooting

### Common Issues

#### Login Issues
- **Problem**: Cannot login to agency dashboard
- **Solution**:
  - Verify email and password are correct
  - Reset password if needed
  - Clear browser cache
  - Try different browser

#### Payment Processing Issues
- **Problem**: M-Pesa payments not processing
- **Solution**:
  - Verify M-Pesa credentials
  - Check API status
  - Verify tenant phone number
  - Contact support if issue persists

#### Commission Calculation Issues
- **Problem**: Commission not calculating correctly
- **Solution**:
  - Verify commission percentage is set correctly
  - Check revenue sharing configuration
  - Verify payment destination
  - Contact support if issue persists

#### Landlord Payout Issues
- **Problem**: Landlord not receiving payouts
- **Solution**:
  - Verify landlord bank account details
  - Check payout schedule
  - Verify commission calculation
  - Contact landlord to confirm details
